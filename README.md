Portfólio Pessoal

Projeto de portfólio desenvolvido com HTML e CSS para apresentar minhas informações, habilidades e projetos na área de desenvolvimento.

🚀 Tecnologias utilizadas

* HTML5
* CSS3

🎯 Objetivo

Criar uma página moderna e responsiva para praticar desenvolvimento web e reunir meus projetos e estudos em um só lugar.

📌 Funcionalidades

* Página inicial personalizada
* Seção sobre mim
* Área de projetos
* Layout responsivo
* Estilização moderna com CSS

📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Estruturação de páginas com HTML
* Estilização com CSS
* Responsividade
* Organização de layout
* Boas práticas no desenvolvimento web

📁 Estrutura do projeto

O projeto está organizado de forma simples e objetiva para facilitar manutenção e futuras melhorias.

🔄 Atualizações futuras

* Adicionar JavaScript
* Melhorar animações
* Inserir novos projetos
* Aprimorar responsividade

⭐ Projeto desenvolvido para aprendizado e evolução na área de desenvolvimento web.
